Example#

First and foremost, the Jupyter Notebook is an interactive environment for writing and running code. The notebook is capable of running code in a wide range of languages. However, each notebook is associated with a single kernel. This notebook is associated with the python kernel, therefor runs python code.

This a test cell for header detection

Header#

There are 3 types of cells:

  • code cells, for R code

  • markdown cells, for nice text

  • raw cells, for text

Code cells allow you to enter and run code#

Double click or use Enter to edit a cell, use Esc to exit and enter what is called command mode

Run a code cell using Shift-Enter or pressing the button in the toolbar above:

weight <- 60
print(weight)

Always run the code you see in these notebooks.

There are two other keyboard shortcuts for running code:

  • Alt-Enter runs the current cell and inserts a new one below.

  • Ctrl-Enter run the current cell and enters command mode.

Important Shortcuts#

Autocomplete#

Try to autocomplete names by hitting Tab instead of typing long names.

Run the cell below:

really_long_variable_name <- 10

Select the next cell, type really and hit Tab

The variable name will be autocompleted, if there is more than one option a menu will pop up

Cells#

Create new cells using the keys a (for above) and b (for below) in command mode or use the button.

Move around the cells with the arrow up arrow down keys or with j and k

You can delete cell by double pressing d

Change the type of a cell with the dropdown below or with:

  • m for markdown

  • y for code

  • r for raw text

Markdown (https://en.wikipedia.org/wiki/Markdown) is a way to add style (bold, headline etc.) to text, e.g typing
